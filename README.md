### Overview
**GangaFlow** is an innovative AI-driven forecasting and monitoring system designed for real-time analysis and intervention in river pollution. Focused on the Ganga River Basin, this project combines state-of-the-art deep learning techniques to empower policymakers, environmental agencies, and stakeholders in preserving one of the world's most vital ecosystems.

---

## üìã Features
- **Real-Time Detection:** Identify and classify polluted regions using drone imagery.
- **Multi-Model Framework:**
  - **YOLOv8:** Bounding box detection for pollution elements.
  - **U-Net:** Semantic segmentation of polluted areas.
  - **AlexNet:** Classification of images as polluted or non-polluted.
- **Image Preprocessing Pipeline:** Includes resizing, normalization, HSV conversion, Gaussian blur, and image augmentations for enhanced model accuracy.
- **Evaluation Metrics:** IoU, F1 Score, Precision, Recall, and Confusion Matrix for robust performance assessment.
